Theme Updates

You can update the theme in two ways are following

  • Automatic Theme updates
  • Manual Theme Updates

Automatic theme updates - using Envato WordPress Toolkit

Envato WordPress Toolkit plugin for automatic WordPress Theme Updates from Themeforest. Themeforest is an reputable theme store they can trust that is know for quality WordPress themes. Many users and developers choose to purchase premium themes from Themeforest. Envato created this plugin so that Themeforest authors can provide buyers with fast, efficient updates to keep their themes bug free and compatible with the latest version of WordPress. This quick guide will show you how to install and setup the Envato WordPress Toolkit plugin.

Before import you need to check the following plugins are installed
  • Revolution Slider
  • Visual Composer
  • night-bird Core
  • Ultimate Addons for Visual Composer

Import XML Files with Importer

Reading the below statement to learn how to manually import the XML file, it will give you all of our demo pages.

  • Log into your WordPress Dashboard.
  • Go to Tools > Import. Then click on WordPress at the bottom of the page.
  • A window should appear click the Install now button.
  • Click Activate plugin & Run Importer.
  • Then choose the themewithwoo.xml file and click upload file and import.
